Detailed Description

template<typename C, typename S, typename V>
class cadmium::celldevs::CellDEVSCoupled< C, S, V >

Abstract base class for coupled Cell-DEVS models.

Template Parameters
Cthe type used for representing a cell ID.
Sthe type used for representing a cell state.
Vthe type used for representing a neighboring cell's vicinities.
